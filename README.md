# ACPICA for 64bit

The branch ACPICA64 is derived from the original **Intel ACPICA project** maintained 
at https://github.com/acpica/acpica **tag** **20251212**.

The ACPICA64 branch contains modifications to support building ACPICA
for 64bit **Windows** and **UEFI** architecture.

The changes reduce compliler warnings at a higher warning level.

Additionally it introduces ACPICA build at the current Visual Studio version VS2026.
